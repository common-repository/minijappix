== Description ==

This plugin add the javascript code for [Jappix mini](http://mini.jappix.com/ "Jappix mini").
You can consider it as a Facebook chat clone based on the XMPP protocol.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload this directory to your plugins directory. It will create a 'wp-content/plugins/minijappix/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
